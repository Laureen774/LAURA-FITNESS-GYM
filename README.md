# LAURA Fitness Gym - Web Application Development

A responsive gym website built with HTML5, CSS3, Bootstrap 5, and JavaScript.

## Pages
1. Home - Hero section + services preview
2. About - Gym story and facility info  
3. Classes - 9 filterable fitness classes 
4. Trainers - Meet our certified coaches
5. Pricing - Membership plans with monthly and annual toggle
6. Schedule - Weekly class timetable
7. Gallery - Photo carousel of gym facilities 
8. Contact - Contact form with validation

## Features
- Fully responsive design using Bootstrap 5 grid
- Consistent theme and navigation across all pages
- JavaScript interactive features: class filter, form validation, pricing toggle
- Real content and images with proper alt text for accessibility
- Tested on Chrome, Firefox, and mobile devices


## How to Run
1. Clone this repo
2. Open `index.html` in your browser
3. No build tools required
